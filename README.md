## Instalação

### Clone do repositório

```bash
$ git clone https://github.com/LuisPeixoto/vidya.git
```

$ cd vidya

````

### Instalação das dependências

```bash
$ yarn install
````

### Execução

```bash
$ yarn android
```

ou

```bash
$ cd ios && pod install
$ yarn ios
```
